LearningAutomata.LRPLearner — Type
LRPLearner(n::Int,
           a::Vector{Float64},
           b::Vector{Float64}, 
           c::Vector{Float64},
           W::Vector{Float64},
           A::Matrix{Float64},
           R::Vector{Matrix{Float64}}, 
           P::Vector{Matrix{Float64}}) <: AbstractLinearLearner

A linear reward-penalty learner.

A general linear reward-penalty learner with n actions, learning rates a for rewards, learning rates b for punishments, action costs c, action probability vector W, vectors of reward and penalty operators R and P, and advantage matrix A.

LearningAutomata.simulate! — Function
simulate!(x::AbstractLearner, iter::Int, c::Vector{Float64})

Simulate a learner for iter iterations in a stationary random environment constituted by penalty probability vector c.

Returns the trajectory as a $m \times n$ matrix, where $m$ is the trajectory length (number of simulation iterations) and $n$ is the learner's dimensionality (number of actions). This can be redirected e.g. to plot in order to visualize the learning trajectory.
